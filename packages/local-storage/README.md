# @character-ui/local-storage

Atomic, integrity-checked per-user filesystem persistence and a cross-process
single-writer lease for Character UI's local npm server and desktop app.

This package is an internal building block of the Character UI monorepo. Node.js
22 or newer is required. See the [project repository](https://github.com/travissavard/Character-UI)
for usage, security policy, and the public `.charui` format.
